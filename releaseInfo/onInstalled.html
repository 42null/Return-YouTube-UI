Thanks for Installing Return YouTube UI!

Steps to enable on all YouTube pages

  1. Navigate to YouTube.
  2. Right click on the extension icon to set permissions.
  3. Select "Always Allow on www.youtube.com".
  4. Reload the page to see changes.

All changes made are local, no data from this extension is ever shared. Nothing is tracked by this extension.
